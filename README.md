# 🧠 LeNet-5: Teste de Conceito (PyTorch)

Este repositório foi criado exclusivamente para testar e demonstrar a implementação da arquitetura **LeNet-5** em PyTorch, utilizando o dataset **MNIST**.

## 📌 Objetivo
Validar o funcionamento de Redes Neurais Convolucionais (CNNs) em um ambiente controlado e simplificado.

## 🛠️ Tecnologias
- **PyTorch**: Framework principal.
- **MNIST**: Dataset de dígitos manuscritos.
- **uv**: Gerenciador de ambiente e dependências.

## 🚀 Como Executar
1. Instale as dependências:
   ```bash
   uv sync
   ```
2. Abra o notebook para ver a implementação:
   `LeNet5_Implementation.ipynb`
